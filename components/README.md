# Components Directory

This directory contains all of the reusable components used throughout the application.

## Subdirectories

-   **`layout/`**: Contains components related to the overall layout of the application, such as the header, footer, and navigation.
-   **`providers/`**: Contains components that provide context to the application, such as the theme provider.
-   **`sections/`**: Contains components that are used to build the different sections of the application's pages.
-   **`shared/`**: Contains components that are shared across multiple pages and sections.
-   **`ui/`**: Contains the application's UI components, such as buttons, cards, and other UI elements.
