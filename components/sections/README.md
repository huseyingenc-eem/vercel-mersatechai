# Section Components

This directory contains components that are used to build the different sections of the application's pages.

## Subdirectories

-   **`ai-otomasyon/`**: Contains components that are used to build the AI automation page.
-   **`hakkimizda/`**: Contains components that are used to build the "About Us" page.
-   **`home/`**: Contains components that are used to build the home page.
