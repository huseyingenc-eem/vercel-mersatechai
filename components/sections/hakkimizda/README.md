# Hakkımızda Section Components

This directory contains components that are used to build the "About Us" page.

## Components

-   **`about-story-section.tsx`**: A section that tells the story of the company.
-   **`core-values-section.tsx`**: A section that highlights the company's core values.
-   **`trust-section.tsx`**: A section that builds trust with potential customers.
-   **`trusted-by-section.tsx`**: A section that showcases the companies that trust and use the company's services.
-   **`why-mersa-section.tsx`**: A section that explains why potential customers should choose the company.
