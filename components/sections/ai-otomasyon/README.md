# AI Otomasyon Section Components

This directory contains components that are used to build the AI automation page.

## Components

-   **`cta-section.tsx`**: The call-to-action section, which encourages users to take a specific action, such as signing up for a demo.
-   **`departments-grid.tsx`**: A grid of departments that can benefit from AI automation.
-   **`problems-section.tsx`**: A section that highlights the problems that AI automation can solve.
-   **`roadmap-section.tsx`**: A section that outlines the company's AI automation roadmap.
-   **`services-catalog.tsx`**: A catalog of the company's AI automation services.
-   **`solution-section.tsx`**: A section that presents the company's AI automation solution.
-   **`use-cases-section.tsx`**: A section that showcases different use cases for AI automation.
-   **`value-proposition-section.tsx`**: A section that explains the value proposition of the company's AI automation services.
