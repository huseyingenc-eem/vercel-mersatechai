# Home Section Components

This directory contains components that are used to build the home page.

## Components

-   **`cta-section.tsx`**: The call-to-action section, which encourages users to take a specific action, such as signing up for a demo.
-   **`features-section.tsx`**: A section that highlights the key features of the company's products and services.
-   **`hero-section.tsx`**: The hero section, which is the first thing users see when they visit the home page.
-   **`integrations-section.tsx`**: A section that showcases the different integrations that the company's products and services support.
-   **`services-section.tsx`**: A section that provides an overview of the company's services.
-   **`use-cases-preview.tsx`**: A section that provides a preview of different use cases for the company's products and services.
