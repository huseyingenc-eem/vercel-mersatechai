# Layout Components

This directory contains components that are used to create the overall layout of the application.

## Components

-   **`footer.tsx`**: The footer component, which is displayed at the bottom of every page.
-   **`logo.tsx`**: The logo component, which is displayed in the header and other parts of the application.
-   **`mobile-menu.tsx`**: The mobile menu component, which is displayed on mobile devices.
-   **`navbar.tsx`**: The navigation bar component, which is displayed at the top of every page.
