# App Directory

This directory contains the core application logic, including all pages, layouts, and API routes.

## Pages

-   **`page.tsx`**: The main landing page of the application.
-   **`ai-otomasyon/page.tsx`**: A page dedicated to AI automation services, showcasing the benefits and features of the company's AI solutions.
-   **`fiyatlandirma/page.tsx`**: The pricing page, which outlines the different service tiers and their costs.
-   **`hakkimizda/page.tsx`**: The "About Us" page, providing information about the company, its mission, and its team.
-   **`hizmetler/page.tsx`**: The main services page, which gives an overview of all the services offered by the company.
-   **`iletisim/page.tsx`**: The contact page, which includes a contact form and other ways to get in touch with the company.

## API Routes

-   **`api/contact/route.ts`**: The API endpoint for handling contact form submissions.

## Layout

-   **`layout.tsx`**: The main layout component for the application, which includes the header, footer, and any other shared UI elements.

## Styling

-   **`globals.css`**: The global stylesheet for the application, which includes base styles and resets.
